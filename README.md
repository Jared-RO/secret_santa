# Secret Santa 🎁

Un proyecto en Python para organizar un intercambio de regalos tipo *Secret Santa*.
Incluye lógica para asignar participantes desde un archivo CSV y enviar notificaciones por correo.

## 📦 Instalación

Clona el repositorio y crea un entorno virtual:

```bash
git clone https://github.com/tu_usuario/secret_santa.git
cd secret_santa
python -m venv venv
source venv/bin/activate  # En Linux/Mac
venv\Scripts\activate     # En Windows
pip install -e .
